# 📚 CA Student Performance Dashboard

## 📌 Project Overview

The CA Student Performance Dashboard is a Power BI analytics project designed to evaluate study habits, stress levels, attendance, and exam outcomes of Chartered Accountancy students. The dashboard provides actionable insights into factors affecting student performance and helps identify at-risk students through interactive visualizations and KPI tracking.

## 🎯 Objectives

- Analyze student performance and academic outcomes.
- Identify factors influencing exam success.
- Monitor stress levels and study patterns.
- Track attendance, revision hours, and mock test performance.
- Support data-driven educational decisions.

## 📈 Dashboard Insights

### Academic Performance
- Evaluated student outcomes using exam scores and pass/fail status.
- Identified performance trends across different CA levels.

### Study Pattern Analysis
- Analyzed relationship between study hours and exam scores.
- Found improvement in scores with increased study duration.

### Student Risk Assessment
- Identified 353 at-risk students requiring academic intervention.
- Highlighted key factors contributing to poor performance.

### Stress & Well-being Analysis
- Measured impact of stress levels on academic performance.
- Examined relationship between sleep hours and exam outcomes.

### Attendance & Revision Analysis
- Compared attendance percentage with final scores.
- Assessed effectiveness of revision hours and mock tests.
